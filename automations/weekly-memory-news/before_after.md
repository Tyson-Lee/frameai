# weekly-memory-news — Before / After

## Before (수동)

매주 월요일 아침, 전략 / 마케팅 인사이트 담당자 1 명이 메모리 4 사 지난
한 주 동향을 모아 임원·유관 부서 공유용으로 한국어 보고서를 작성한다고
가정합니다.

| 작업 | 평균 소요 |
|---|---|
| 회사별 검색 (구글 / 네이버 / 야후 재팬) — 4 사 × 영·한·일 쿼리 분산 | 20-30 분 |
| 헤드라인 훑기 + 페이월·중복 기사 제거 + 본문 정독 | 30-45 분 |
| 회사별 주요 발표 정리 (3-6 개 / 사) + 출처 URL 정리 | 25-35 분 |
| 임원용 1 페이지 요약 작성 (정중한 한국어) | 25-40 분 |
| 출처 다시 확인 (URL 끊김·페이지 변경) + 인용 체크 | 10-15 분 |
| **합계** | **110-165 분 (≈ 2-3 시간)** |

> 기준: 한 주 동안 4 사 합산 12-24 개의 보도/발표가 있는 평범한 주.
> 신제품 발표·실적 발표가 겹치는 주는 3-4 시간까지 늘어남.

## After (자동)

```bash
./frame run weekly-memory-news "월요일 자동 실행"
```

| 작업 | 평균 소요 |
|---|---|
| `frame run` 호출 + helper (`prepare.py`) 실행 | 5-10 초 |
| Claude 가 4 사 × WebSearch fan-out (3-9 쿼리) | 30-60 초 |
| 후보 hit 들 WebFetch 로 본문 추출 (총 12-24 페이지) | 60-120 초 |
| executive_summary + 회사별 4 파일 작성 + helper concat | 60-90 초 |
| 별도 컨텍스트 auditor 의 출처·인용 검증 (refute-first) | 30-60 초 |
| 사용자가 결과 검토 + 헤드라인 spot-check | 5-10 분 |
| **합계** | **8-15 분 (모델 실행 ≈ 3-5 분 포함)** |

> 모델 실행 시간은 WebSearch / WebFetch 응답 지연에 가장 크게 좌우됨.
> 한 주 보도량이 많을수록 fetch 단계가 길어지지만, 4 사 병렬이라 회사
> 수에 선형 비례하지는 않음.

## 무엇이 자동화되지 않는가 (정직)

- **사내 정보 결합은 사람 몫**. 외부 뉴스만 다룹니다. 자사 (Samsung
  메모리 등) 의 비공개 진척과 경쟁사 동향을 묶어 인사이트로 만드는
  단계는 자동화 범위 밖입니다.
- **유료 리서치 통합 없음**. Gartner / Counterpoint / TrendForce /
  Yole 의 페이월 리포트는 자동으로 가져오지 못합니다.
- **검토 단계는 사람**. refute-first auditor 가 출처 URL / 원문 인용
  매칭을 한 번 확인하지만, "이 보도가 사실상 동일 사건의 재포장인가",
  "이 추정치는 신뢰할 만한가" 같은 메타 판단은 사람이 들고 있어야
  합니다.
- **임원 보고 표지 / 슬랙 / 이메일 발송**. 결과는 마크다운 파일만
  생성. PowerPoint 변환, Slack 게시, Confluence 업로드는 별도 스킬·
  수동 작업입니다.
- **검색 엔진 편향 보정 없음**. 영어 매체에서만 보도된 사건과 한국어
  매체에서만 보도된 사건의 가중치를 따로 보정하지 않습니다 — 양 언어
  쿼리를 fan-out 하는 것으로 끝.
- **품질은 한 주의 보도량에 비례**. 한 주가 조용하면 보고서도
  조용합니다 ("이번 주 공개된 메모리 관련 발표 없음"). 허위 보강 없음.

## 비용 (대략)

- Anthropic API 호출: 1 회 실행당 약 **$0.30 – $1.20**
  (WebSearch / WebFetch 호출 + Sonnet/Opus 추론 + refute-first auditor 1 회 포함).
- 주 1 회 사용 시 연간 약 **$16 – $63**.

수동 작업 (한 회당 2-3 시간, 시간당 $40 기준) 의 약 **80-100 시간 / 연** 을
$16-63 / 연 으로 대체하므로 ROI 는 1,500-25,000 배 수준.
